5 elementos: agente + ação + modo + finalidade + efeito. Respeitar direitos humanos.

Estrutura
01

Introdução (5–7 linhas)

Repertório sociocultural + contextualização + tese clara. Nunca comece com "Desde os primórdios..."

02

1º Desenvolvimento (8–10 linhas)

Tópico frasal + 1º argumento + dados/exemplos concretos + mini-conclusão.

03

2º Desenvolvimento (8–10 linhas)

Argumento diferente do 1º + análise crítica + operador argumentativo.

04

Conclusão (5–7 linhas)

Retomada da tese + proposta de intervenção com os 5 elementos obrigatórios.
